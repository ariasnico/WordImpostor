🚀 INICIO RÁPIDO (3 PASOS)

1. Abre una terminal en este directorio

2. Ejecuta según tu sistema operativo:

   Linux/Mac:
   ./START.sh

   Windows:
   START.bat

3. El navegador se abrirá automáticamente en http://localhost:3000

¡Listo! Ya puedes jugar.

═══════════════════════════════════════════════════════════════════

📱 CÓMO JUGAR

1. Ingresa tu nombre
2. Crea una sala (serás el host) o únete con un código
3. Comparte el código de 6 dígitos con tus amigos
4. El host inicia el juego cuando todos estén listos (mínimo 2)
5. Sigue las instrucciones en pantalla:
   - Un jugador elige la palabra secreta
   - Recibes tu rol (Crewmate o Impostor)
   - Da pistas en tu turno
   - Vota al sospechoso
   - ¡Gana!

ROLES:
- 👥 Crewmates: Conocen la palabra, deben identificar al impostor
- 🎭 Impostor: No conoce la palabra, debe fingir que sí

═══════════════════════════════════════════════════════════════════

🔧 SOLUCIÓN DE PROBLEMAS

Si el script automático no funciona:

1. Verifica que Node.js esté instalado:
   node --version
   (Debe mostrar v14 o superior)

2. Instala las dependencias manualmente:
   npm run install-all

3. Inicia el servidor en una terminal:
   cd server
   npm start

4. Inicia el cliente en otra terminal:
   cd client
   npm start

Para más ayuda, lee QUICKSTART.md

🌐 JUGAR EN RED LOCAL

Para jugar con amigos en la misma WiFi:

1. Encuentra tu IP local:
   - Linux/Mac: ifconfig | grep "inet "
   - Windows: ipconfig

2. Comparte tu IP con tus amigos, ejemplo:
   http://192.168.1.100:3000

3. Ellos escanean el código QR o ingresan manualmente
